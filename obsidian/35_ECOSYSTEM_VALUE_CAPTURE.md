# Ecosystem Value Capture

Use this note when thinking about:

- treasury funding
- ecosystem grants
- retroactive public goods funding
- protocol partnerships
- getting paid for driving usage to chains or protocols

Detailed source:

- [Ecosystem Funding And Measurement Map](../docs/ECOSYSTEM_FUNDING_AND_MEASUREMENT_MAP.md)

## Core rule

Ecosystems do not pay because a product sounds important.

They pay when a product can prove:

- user adoption
- attributable protocol usage
- retained activity
- strategic value to the ecosystem

## ChopDot implication

ChopDot should be built so it can later show:

- commitments created
- commitments completed
- users brought into a chain/protocol flow
- value or activity routed through a specific ecosystem
- repeat usage and retained coordination
